# Lobo Cinzento 🐺🔔

European Portuguese, delivered by the bell. A phone-first PWA: four short sessions a day arrive as calendar events on your phone — input at dawn, drills at noon, a spoken conversation in the afternoon, a test at night. One tap on the event drops you into the session. The bell removes excuses, never effort.

- Comprehensible-input curriculum, frequency-ordered, European Portuguese throughout
- Socratic interleaved drilling that hunts weak spots
- Earned access: read Lição 0, pass the gate, get your calendar
- Static site, no accounts, no tracking; progress lives in your browser

Built with no dependencies and no build step. Deployed on GitHub Pages.
